1. What are the new tags added in HTML5? HTML5 introduced several new tags to enhance the structure and semantics of web pages. Some of the key new tags include:
: Defines a header for a document or a section.
: Represents a section of a page intended for navigation links.
: Defines a section in a document, often with a heading.
: Specifies independent, self-contained content.
: Represents content somewhat related to the main content, such as sidebars.
: Defines a footer for a document or a section.
